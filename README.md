# рџ’ё Expenzo - Full Stack Expense Tracker

Expenzo is a modern full-stack expense tracker web application built using React, Flask, and MongoDB. It helps users manage daily expenses with smart analytics, filters, and interactive visual insights. The application features a responsive dashboard, expense tracking, monthly overview, and secure user authentication.

Users can:
- Register & Login
- Add Expenses
- Update Expenses
- Delete Expenses
- Filter Expenses
- View Analytics
- Track Monthly Spending
- Manage Personal Expenses Smartly

# рџ”Ґ Backend Setup

## Go to backend folder

```bash

```

## Create virtual environment

```bash
python -m venv venv
```

## Activate virtual environment

### Windows

```bash
venv\Scripts\activate
```

### Mac/Linux

```bash
source venv/bin/activate
```

## Install dependencies

```bash
pip install -r requirements.txt
```

## Create `.env`

```env
MONGO_URI=your_mongodb_uri
SECRET_KEY=your_secret_key
```

## Run Backend

```bash
python app.py
```

Backend runs on:

```bash
http://127.0.0.1:5000
```
